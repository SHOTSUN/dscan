jQuery-viewport-checker
https://www.drupal.org/project/viewportchecker
==============================================

DESCRIPTION
-----------
jQuery-viewport-checker is a Drupal 7 integration of jQuery-viewport-checker library. Little script that detects if an element is in the viewport and adds a class to it.

DEPENDENCIES
------------
Libraries
jquery.viewportchecker.min.js in sites/all/libraries/jquery.viewportchecker (can be downloaded from https://github.com/dirkgroenen/jQuery-viewport-checker)

INSTALLATION AND CONFIGURATION
------------------------------
1) Download and add the library (need 3 files: jquery.viewportchecker.js, jquery.viewportchecker.min.js and package.json) 
from https://github.com/dirkgroenen/jQuery-viewport-checker in sites/all/libraries/jquery.viewportchecker
2) Install the module as usual.
3) Enable the module
4) In your custom js file you can write the code. Library usage: https://github.com/dirkgroenen/jQuery-viewport-checker/blob/master/README.md
